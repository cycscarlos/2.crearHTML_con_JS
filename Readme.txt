Javascrip Learning. Seriado #2.                   

Contenido: "Como modificar las propiedades CSS de un elemento con Javascript"
------------------

1. Se crearán tres botones y una imagén con Javascript directamente (no con HTML)
2. Botón 1: Al hacer click se cambian las propiedades de la imagén y se oculta el botón
3. Botón 2: Al hacer click la imagén desaparece/aparece conforme se vaya pulsando dicho botón
4. Botón 3: al hacer click, se restablecen todas las propiedades como estaban inicialmente

Temática:
--------

1. Eventos | funciones | selectores | innerHTML | insertAdjacentHTML
2. innerHTML | value | substr | toLowerCase | toUpperCase
3. value | funciones | random | floor | onkeydown | key | number
4. variables | funciones | expresiones | operadores | if | else | style
5. querySelector vs getElementById
